# Awesome Shell

Uma lista criteriosamente selecionada de excelentes frameworks, kits de ferramentas, guias e utilitários para linha de comando. Inspirado pelo *awesome-php*, este repositório serve como um recurso abrangente para desenvolvedores e administradores de sistema que desejam aprimorar seu ambiente de terminal e produtividade.

> **Nota:** Esta coleção incrível também está disponível em [Unix-Shell.ZEEF.com](https://unix-shell.zeef.com/caleb.xu).

## Redes e Servidores

*Servidores leves auto-hospedados e ferramentas de rede escritas em scripts de shell.*

* [balls](https://github.com/jneen/balls) - Bash on Balls
* [bashblog](https://github.com/cfenollosa/bashblog) - Um script Bash que gerencia postagens de blog
* [bashttpd](https://github.com/avleen/bashttpd) - Um servidor web escrito em bash
* [bitpocket](https://github.com/sickill/bitpocket) - "DIY Dropbox" ou "diretório bidirecional (r)sync com exclusão adequada"
* [dokku](https://github.com/progrium/dokku) - Mini-Heroku baseado em Docker em cerca de 100 linhas de Bash
* [Dropbox-Uploader](https://github.com/andreafabrizi/Dropbox-Uploader) - Um script BASH que pode ser usado para fazer upload, download, listar ou deletar arquivos do Dropbox
* [httpie](https://github.com/jakubroztocil/httpie) - Um cliente HTTP de linha de comando, substituto amigável para o cURL
* [ngincat](https://github.com/jaburns/ngincat) - Servidor HTTP minúsculo em bash usando netcat
* [pushblast](https://github.com/alebcay/pushblast) - Receba notificações do PushBullet quando um programa de shell terminar
* [pushbullet-bash](https://github.com/Red5d/pushbullet-bash) - Interface Bash para a API do PushBullet
* [resty](https://github.com/micha/resty) - Pequeno cliente REST de linha de comando que você pode usar em pipelines
* [aria2](https://github.com/tatsuhiro-t/aria2) - Um utilitário de download leve e multi-plataforma operado na linha de comando
* [youtube-dl](https://github.com/rg3/youtube-dl) - Pequeno programa de linha de comando para baixar vídeos do YouTube e outros sites

* [bash-git-prompt](https://github.com/magicmonty/bash-git-prompt) - Um prompt bash informativo e elegante para usuários Git
* [bash-powerline](https://github.com/riobard/bash-powerline) - Prompt Bash com estilo Powerline em script Bash puro
* [bashstrap](https://github.com/barryclark/bashstrap) - Uma maneira rápida de embelezar o terminal do OSX
* [flatui-terminal-theme](https://dribbble.com/shots/1021755-Flat-UI-Terminal-Theme) - Cores mais agradáveis para terminal
* [git-extras](https://github.com/tj/git-extras) - Utilitários Git -- resumo de repositório, rebasing, geração de changelog e outras belezas

## Gerenciamento de Pacotes e Frameworks

*Frameworks e gerenciadores de pacotes para scripts de shell.*

* [bash-it](https://github.com/Bash-it/bash-it) - Uma estrutura comunitária para Bash inspirada no Oh My Zsh
* [basher](https://github.com/basherpm/basher) - Um gerenciador de pacotes para scripts de shell
* [bpkg](https://github.com/bpkg/bpkg) - Gerenciador de pacotes leve para bash
* [dotzsh](https://github.com/dotphiles/dotzsh) - Um framework para gerenciar sua configuração zsh
* [oh-my-fish](https://github.com/oh-my-fish/oh-my-fish) - O Framework para o Fish Shell
* [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh) - Um framework delicioso, open source e impulsionado pela comunidade para gerenciar sua configuração Zsh
* [prezto](https://github.com/sorin-ionescu/prezto) - O framework de configuração para Zsh
* [shallow](https://github.com/adrianscheff/shallow) - Um gerenciador de dependências simples para scripts de shell

## Escrita e Teste de Scripts

*Ferramentas para escrever, depurar e testar scripts de shell.*

* [shellcheck](https://github.com/koalaman/shellcheck) - Uma ferramenta de análise estática para scripts de shell
* [shunit2](https://github.com/kward/shunit2) - Um framework de teste unitário para scripts Bash com suporte a ksh
* [bats](https://github.com/bats-core/bats-core) - Sistema de Teste Automatizado Bash (Bash Automated Testing System)
* [rebash](https://github.com/jandob/rebash) - Biblioteca/Framework de scripting para bash
* [shfmt](https://github.com/mvdan/sh) - Um analisador, formatador e interpretador de shell com suporte a bash
* [semver_bash](https://github.com/stephenc/semver_bash) - Versionamento semântico para scripts Bash

## Multimídia

*Ferramentas para gerenciar imagens, áudio e vídeo da linha de comando.*

* [animbar](https://github.com/dylanaraps/animbar) - Um script para corrigir proporções de vídeo
* [album-art](https://github.com/antznin/album-art) - Baixe capas de álbuns pela linha de comando
* [clipview](https://github.com/dylanaraps/clipview) - Visualize imagens no terminal usando arte ASCII
* [ffmpeg-bash](https://github.com/kristian-l/ffmpeg-bash) - Wrapper Bash para ffmpeg
* [gifgen](https://github.com/lukechilds/gifgen) - Um gerador de GIF simples
* [imgp](https://github.com/jarun/imgp) - Um visualizador de imagens rápido e moderno para CLI, além de redimensionador/rotacionador em lote
* [mpd-mpris](https://github.com/nicolasbrugneaux/mpd-mpris) - Plugin MPRIS para MPD
* [pianobar](https://github.com/Promo/pianobar) - Reprodutor de música da Pandora.com via linha de comando
* [readme-video](https://github.com/dylanaraps/readme-video) - Um script para exibir vídeos em arquivos markdown
